Project 01 — Brand Translation
Ambria College of Nursing
Tuition & Fees Page
View Live
WordPress Blocks HTML / CSS Brand Adaptation Content Migration

When Judson University acquired Ambria College of Nursing, Ambria's tuition and fee information existed only inside Judson's academic catalog — a dense, table-heavy page with no Ambria branding or visual hierarchy. The goal was to give that content a proper home on Ambria's own site: same information, completely different experience.

Key Decisions
01
Hero image for context
The catalog page had no visual identity. Adding a header image immediately places the user in Ambria's world and signals this is a distinct institution, not just another Judson catalog page.
02
Payment dates surfaced visually
In the catalog, payment deadlines were buried in a bullet list. The redesign lifts them into a three-column visual block — the most time-sensitive information is now the first thing a student sees.
03
Ambria brand, not Judson's
Color, type, and nav all follow Ambria's identity. Students coming to Ambria's site shouldn't feel like they've accidentally landed on Judson University's catalog.
04
Cleaner table hierarchy
The source tables had inconsistent formatting and no clear section boundaries. Each fee category is now visually distinct, making it easier to scan for what you need.
Project 02 — From Scratch Design
Judson University
News & Events Page
View Live
WordPress / Beaver Builder HTML / CSS Information Architecture From Scratch

Judson's existing news presence was scattered — individual department pages, no central hub, and no consistent way for the university to surface events, announcements, and press releases in one place. This page was designed and built from scratch to serve as that hub, organized around how different types of content actually get consumed.

The Page
judsonu.edu / news-and-events
Add full-page screenshot of the live page
Information Architecture Decisions
01
Five content types, one page
Events, news stories, announcements, media releases, and community submissions all live here. The challenge was giving each category visual space without the page feeling like a dump.
02
Sticky jump navigation
A sticky section nav lets users jump directly to what they need. Someone checking for an upcoming event doesn't need to scroll past news stories to get there.
03
Urgency in announcements
Announcements are color-coded and badged by type — urgent deadlines, campus updates, IT notices. Critical information is visually separated from general updates.
04
Submission CTA built in
Faculty and staff need a way to submit news and events. Rather than sending them elsewhere, the submission pathway lives directly on the page — closing the loop.
05
Judson's brand, faithfully applied
Navy, gold, and Open Sans follow Judson's existing design system. The page needed to feel native to the site, not like a new addition bolted on from outside.
06
Built in WordPress natively
No page builder plugins — this was built directly in WordPress using blocks and custom HTML/CSS, keeping it maintainable for non-technical staff after handoff.
